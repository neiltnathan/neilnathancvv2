Access Management Request Application

Description:

The Access Management Request Application was developed at the request of a corporate customer to automate the access requests to Lotus Notes applications, to document (audit) the request processes and to eliminate as many manual process steps as possible.

Key Features:

  • A fully automated Lotus Notes access request application for end users to request different levels of access to one or more Lotus Notes applications.
  • End users can request access to a single or multiple Lotus Notes applications
  • Notifications are sent to application owners upon access request for approval
  • Upon approval access is granted autmatically to the requesed application
  • An audit log records all access request activity.

Key Achievements:

  • Cost effective.
  • Efficient handling of requests

Archive Application

Description:

The Lotus Notes Archive application was developed at the request of a corporate customer to help manage the number of Lotus Notes applications in production.

Key Features:

  • A fully automated Lotus Notes archiving application
  • Archive application monitors change of archive status for each application
  • Upon an archive request the archive process is initiated
  • After 30 days the application is automatically transferred to tape backup
  • Archived application is automatically removed from production server
  • An audit log records all access request activity.

Key Achievements:

  • Cost effective.
  • Fully automated
